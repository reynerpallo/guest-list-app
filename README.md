# Guest List Application

Aplikasi sederhana untuk mencatat nama tamu yang berkunjung ke portofolio.

## Fitur
- Tambah nama tamu
- Lihat daftar tamu

## Cara Menjalankan
1. Clone repo ini:
    ```bash
    git clone <URL_REPO>
    ```
2. Pindah ke folder proyek:
    ```bash
    cd guest-list-app
    ```
3. Jalankan program:
    ```bash
    python main.py
    ```
